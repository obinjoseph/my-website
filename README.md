# Obin Joseph — Portfolio Website

Personal portfolio website for **Obin Joseph**, a Data Engineer based in Newcastle, UK. The site is a lightweight static website built with plain HTML, CSS, and JavaScript and highlights experience, tech stack, certifications, and contact information.

## Live site

- Primary URL: `https://www.obinjoseph.com`

## Overview

This project is a single-page portfolio with the following sections:

- **Hero** introduction with primary calls to action
- **About** section with experience summary and profile details
- **Tech Stack** grouped by languages, databases, infrastructure, analytics, frameworks, and design tools
- **Certifications** with external verification links
- **Contact** form connected to Google Forms, plus LinkedIn

## Features

- Responsive single-page layout
- Animated canvas background with reduced-motion support
- Sticky navigation with active section highlighting
- Reveal-on-scroll transitions using `IntersectionObserver`
- Dynamic calculation of years of experience in the About section
- Collapsible Tech Stack and Certifications sections with floating in-section toggles
- Contact form submission through Google Forms with toast feedback
- Performance-oriented loading for fonts, stylesheet, and icon assets
